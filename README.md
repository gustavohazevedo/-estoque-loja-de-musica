# Estoque - Loja de Música

Sistema básico em Python para gerenciar o estoque de uma loja musical através do terminal.

## 🛠️ Funcionalidades

* **1** - Cadastrar novo produto.
* **2** - Listar todos os produtos.
* **3** - Buscar produto por ID.
* **4** - Atualizar quantidade no estoque.
* **5** - Sair do programa.

## 🚀 Como rodar

1. Baixe o arquivo de código (ex: `estoque.py`).
2. Abra o terminal na pasta do arquivo.
3. Execute o comando:
   ```bash
   python estoque.py
   ```
